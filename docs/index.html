Introduction

As Large Audio-Language Models (LALMs) advance, the ability to generate speech with appropriate paralinguistic information (tone, emotion, rhythm, etc.) becomes increasingly important. However, there lacks a comprehensive benchmark to evaluate this capability.

SpeechParaling-Bench addresses this gap by providing:

  • Three Core Evaluation Tasks: Paralanguage Control, Dynamic Variation, and Situational Adaptation
  • Bilingual Coverage: Chinese-English aligned prompts across 5 life scenarios (Daily Life, Campus, Workplace, Family, Entertainment)
  • Fair Comparison: Baseline-candidate evaluation protocol using Gemini-3 Pro as judge
  • Comprehensive Dimensions: Covers Expressive, Prosodic, and Acoustic paralinguistic features

Paralinguistic Feature Categories

We categorize 13 paralinguistic dimensions into three main categories:

Expressive Features

Emotional and affective dimensions that convey feelings and attitudes

Emotion Attitude Style

Prosodic Features

Rhythmic and tonal elements of speech delivery

Pitch Pace Pause Rhythm Volume Stress

Acoustic Features

Voice quality and spectral characteristics

Age Timbre Cognitive State Non-Linguistic Vocalizations
