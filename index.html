Introduction

As Large Audio-Language Models (LALMs) advance, the ability to generate speech with appropriate paralinguistic information (tone, emotion, rhythm, etc.) becomes increasingly important. However, there lacks a comprehensive benchmark to evaluate this capability.

SpeechParaling-Bench addresses this gap by providing:

  • Three Core Evaluation Tasks: Paralanguage Control, Dynamic Variation, and Situational Adaptation
  • Bilingual Coverage: Chinese-English aligned prompts across 5 life scenarios (Daily Life, Campus, Workplace, Family, Entertainment)
  • Fair Comparison: Baseline-candidate evaluation protocol using Gemini-3 Pro as judge
  • Comprehensive Dimensions: Covers Expressive, Prosodic, and Acoustic paralinguistic features

Evaluation Pipeline

We employ a fully automatic, transcription-free evaluation framework that directly assesses generated audio using a Large Audio-Language Model (LALM) as a judge.

1

Data Engine

Leverages Gemini to synthesize textual instructions based on 100+ dimensions, followed by IndexTTS to generate high-quality audio prompts.

2

Pairwise Comparison

Conducts blind tests between baseline and candidate models using randomized ordering to mitigate positional biases.

3

LALM-as-a-Judge

Gemini-3.0-Pro evaluates responses with Chain-of-Thought reasoning, grounded in specific audio timestamps.

Insights & Failure Analysis

Multilingual Gap: Leading models still struggle to maintain rich paralinguistic features and human-like fluency across multiple languages simultaneously.
Static vs. Dynamic: Balancing static control (Paralanguage Control) with temporal dynamics (Dynamic Variation) remains a significant challenge for modern LALMs.
Implicit Bottleneck: 43.3% of errors in situational dialogues stem from the model's failure to capture paralinguistic cues hidden in user speech.
